The GraphBuilder utility is designed to work with gephi to do the following:
- Generate graph files
Example Using GUI:
- Open the utility
- Select the directory ex: "GephiFiles" that contains folders where each folder contains a nodes.csv and edges.csv file.
- If your nomenclature differs enter your node file name and your edge file name.
- click the button to build the graph
Example Using Headless:
- Run the GraphBuilder.jar with the command line arg '--help' and a detailed syntax and usage will be displayed.
